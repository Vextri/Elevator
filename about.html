Project VI - Elevator Project
ESE 2026
Amy Wentzell, Blaise Swan

Project Overview

The purpose of this project is to implement a fully functioning elevator, using
CANBus and wireless communications. This website is for project documentation,
including our logbooks, project plan, test plan(s), and the final GUI for
controlling the elevator online.

Objectives

  • Implement a working elevator system, using a fully functioning CAN network
  • Implement a data repository and web-based GUI to control the elevator
  • CAN diagnostics, logs
  • Testing, analysis, and successful implementation of system

Project Materials

  • Rapberry Pi 5
  • Arduino Uno
  • STM32 F303RE Nucleo-64 Microcontroller boards (4)
  • Pre-fabricated Elevator with CANBus connectors and Nucleo Shields
