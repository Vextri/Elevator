Explanation of Basic HTML Tags

1. <!DOCTYPE html>

The <!DOCTYPE html> declaration defines the document type and version of HTML. It is placed at the very top of the page to let the browser know that it is working with HTML5.

2. <html> tags

The <html> tag surrounds all the content of the HTML page.

3. <body> tags

The <body> tag surrounds everything that is shown in the browser.

4. <head> tags

The <head> tag contains meta-information about the page (like the title, styles, scripts, etc.).

5. <title> tags

The <title> tag sets the title shown on the browser tab.

6. <h1> to <h6> tags

The <h1> to <h6> tags are used for headings, from largest to smallest in size and importance.

7. <p> tags

The <p> tag defines a paragraph and usually adds spacing between blocks of text.
